## Highlights


## Installation

```sh
# with yarn:
yarn add ship-7-api-lib

# or with npm: 
npm install --save ship-7-api-lib
```


## Usage

```js
import ship7ApiLib from 'ship-7-api-lib';

// do what you wanna do
console.log(ship7ApiLib);
```
